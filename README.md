# Warhammer 40,000 — Tabletop Prototype

A browser-based Warhammer 40K 10th Edition tabletop simulator. Play solo or 1v1 online with a friend.

**[▶ Play now](https://thomasrieger.github.io/40K-game)**

---

## Setup

### Build Your Team
1. Click **⚙ Edit Teams** on the main menu
2. Pick your faction (Space Marines or Orks)
3. Add units from the shop — each has real 10E codex stats
4. Stay within the **500 pt budget**
5. Switch to the **Player 2** tab to set up the second team (local play)
6. Click **Save & Close**

### Play
- Click **⚔ Play** to start the game

---

## Online 1v1

1. Both players open **[the game](https://thomasrieger.github.io/40K-game)**
2. Each player sets up their team in **Edit Teams → Player 1 tab** (your team)
3. **Host**: click 🌐 Online → **Host** → copy the room code
4. **Guest**: click 🌐 Online → **Join** → paste the code → connect
5. Game starts automatically — host is Player 1, guest is Player 2

---

## How to Play

The game follows standard Warhammer 40K turn structure. Each turn has 5 phases:

| Phase | What you can do |
|-------|----------------|
| **Command** | Spend 1 CP on an ability (Heal +2 HP or Dash +2.5") |
| **Movement** | Move a unit up to its M value |
| **Shooting** | Shoot an enemy in range and line of sight |
| **Charge** | Rush an enemy within 12" |
| **Fight** | Melee attack an adjacent enemy |

Click **Next Phase ▶** to advance. At the end of the Fight phase, the turn passes to the other player.

### Controls

| Action | How |
|--------|-----|
| Select unit | Left-click your unit |
| Move / Shoot / Charge / Fight | Select unit → click action button → click target |
| Pan camera | Middle-click drag or right-click drag |
| Zoom | Scroll wheel |
| Inspect enemy | Hover over enemy unit |
| Select multiple units | Ctrl+click or Ctrl+drag to box-select |
| Group action | Select 2+ units → use group panel buttons |
| Deselect | Click empty space |

### Combat
Dice rolls follow 10E rules:
1. **Hit** — roll attacks vs Weapon Skill (melee) or Ballistic Skill (ranged)
2. **Wound** — successes roll vs Strength/Toughness table
3. **Save** — defender rolls armour save (modified by AP)
4. Damage applied per failed save × weapon Damage stat

### Unit Stats

| Stat | Meaning |
|------|---------|
| M | Move — inches per phase |
| T | Toughness — harder to wound |
| SV | Save — roll this or higher to block damage |
| W | Wounds — total HP |
| LD | Leadership |
| OC | Objective Control |

### Win Condition
Eliminate all enemy units.
